### 2. `frontend/src/index.css` вЂ” Invalid Tailwind class removed

**Before:**
```css
* { @apply border-border; }
```
`border-border` does not exist in Tailwind. Caused a PostCSS hard crash on `npm run dev`.

**After:**
```css
* { box-sizing: border-box; }
```

`ClockArrowUp` does not exist in the installed version of `lucide-react`. Replaced with `Clock` in import and all JSX usages.

### 4. `frontend/src/components/layout/Layout.jsx` вЂ” Same icon fix
